Garage-PARROT
https://garageapp.fly.dev/main

## Installation
Les instructions pour installer et configurer le projet.

Build:
     > npm install
Run:
     > npm start

## Utilisation

1-client: peut filter les voitures, envoyer une message par une formulaire de contacts et mettre une temoignage.
2-proprietaire: peut se connecter comme adminstrateur, ajouter un employe et ajouter/modifier les services les horaires.
3-employe : peut ajouter une voiture d'occasion et controler les temoignages.
